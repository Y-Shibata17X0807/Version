案件フォルダ自動生成ツールの使用方法

1.工番名を入力して下さい

注意事項
1.1.発年度（XまたはZの前2ケタ）は必ず2桁で入力して下さい
例：2018年度だったら、18Xまたは18Z
　　2001年度だったら、01Xまたは01Z

1.2.開発年度の後は、X（大文字小文字どちらも可）またはZ（大文字小文字どちらも可）を必ず入力して下さい。
※：XまたはZが小文字だった場合、アプリが自動的に大文字に置き換えて、フォルダを作成します。


1.3.XまたはZの後は、必ず-（ハイフン）を入力して下さい


1.4.ハイフンの後は、必ず4桁の整数で入力して下さい
例：通し番号が100番目だった場合
　　0100と4桁で入力
　　
　　


2.顧客名称を入力して下さい
例：羽野製作所 開発部
会社名以下を入力する際は、"半角スペース"で隙間を開けて入力して下さい



3.案件名称を入力して下さい
例：案件フォルダ自動生成ツール
案件名に"改版"などが付く場合は"半角スペース"で隙間を開けて入力して下さい。



既定値で作成されるフォルダの一覧を下記に記します。

開発工番（顧客名 案件名）←親フォルダ
 |_ 00_製作伝票　        ←子フォルダ
 |_ 01_見積もり
 |_ 02_客先資料
 |_ 03_提出資料
 |_ 04_仕様
 |_ 05_検討
 |_ 06_部材発注
 |_ 07_客先打合せ
 |_ 20_回路設計
 |_ 30_機構設計
 |_ 40_ソフト設計
 |_ 50_出荷
 |_ 60_稟議
 |_ 99_ワークフロー
 
 
 
 
 デフォルト以外のフォルダを追加したい場合
 １．メモ帳起動して下さい
 ２．下の注意書きに従って、Key名とValue名を入力して下さい（例：0 = 00_製作伝票）
 ３．カスタムが終了しましたら、ファイル名を「FolederSetting.ini」として、任意の場所に保存して下さい。

 ※：ただし、作成できるフォルダの数は最大３０個までです。
 
 
・フォルダ名は、以下のような書式で読み取られます
→Key名 = Value名

なので、追加する時は
→14 = 100_その他

の書式で追加していただくと、正常に追加されます

注意：Key名は、整数の連番（1,2,3,4,5…）で設定して下さい。
　　　英文字（a,b,c,d,eやA,B,C,D,E）や日本語では、エラーが出ます。
